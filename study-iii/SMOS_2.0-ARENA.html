Release Note for SMOS 2.0

Improvements

  • SMOS4 Course of Studies not shown in alphabetic order

    • Modified method getCourseOfStudyList() in class ManagerCourseOfStudy.

  • SMOS9 Implement the Singleton design pattern

    • New method getInstance() in ManagerClassroom. Modified methods ManagerClassroom(), delete(Classroom), getClassroomsByStudent(User), getClassroomByUserAcademicYear(User,int), and getClassroomsByTeacher(User).

    • New method getInstance() in ManagerCourseOfStudy. Modified methods ManagerCourseOfStudy(), delete(CourseOfStudy), assignTeachingAsCourseOfStudy(CourseOfStudy,Teaching), and removeTeachingAsCourseOfStudy(CourseOfStudy,Teaching).

    • New method getInstance() in ManagerRegister. Modified methods ManagerRegister(), getRegisterByClassIDAndDate(int,Date), and getRegisterLineByClassIdDateUserId(int,Date,int).

    • New method getInstance() in ManagerTeaching. Modified methods ManagerTeaching(), delete(Teaching), and loadRecordFromRs(ResultSet).

    • New method getInstance() in ManagerUser. Modified methods ManagerUser(), and delete(User).

    • New method getInstance() in class ManagerVotes.

    • Modified method doGet(HttpServletRequest,HttpServletResponse) in classes ServletDeleteClassroom, ServletInsertClassroom, ServletShowClassroomDetails, ServletShowClassroomList, ServletAddRemoveTeachingsAsCourseOfStudy, ServletCourseOfStudyTeachings, ServletDeleteCourseOfStudy, ServletShowCourseOfStudyDetails, ServletDeleteJustify, ServletDeleteNote, ServletInsertAbsenceAta, ServletInsertDelayAta, ServletInsertJustify, ServletInsertNewNote, ServletLoadClassForTeacher, ServletShowClassroomManagement, ServletShowJustifyDetails, ServletShowJustifyList, ServletShowNoteDetails, ServletShowNoteList, ServletShowRegister, ServletUpdateJustify, ServletDeleteReport, ServletInsertReport, ServletLoadReport, ServletLoadTeachingList, ServletShowClassroomByAcademicYear, ServletShowReports, ServletShowReportStudent, ServletShowStudentByClassForTeacher, ServletShowStudentsByClass, ServletUpdateReport, ServletDeleteTeaching, ServletShowTeachingDetails, ServletShowTeachingList, ServletUpdateTeaching, ServletAddTeachingAsTeacher, ServletAlterPersonalDate, ServletAssignRole, ServletAssignStudentClassroom, ServletDeleteUser, ServletInsertUser, ServletLoadClassByAccademicYear, ServletLoadYear, ServletLogin, ServletRemoveStudentClassroom, ServletRemoveTeachingAsTeacher, ServletReportTeachings, ServletShowTeacherTeachingFormByClass, ServletShowUserDetails, ServletShowUserList, ServletShowUserRoleForm.

  • SMOS10 Days between dates is useless

    • Method daysBetween(Date,Date) in Utility has been deprecated.

New Features

  • SMOS2 Add the field "Description" to the Course of Studies

    • New field description of type String in CourseOfStudy. New methods getDescription(), and setDescription(String) in the same class.

    • Modified methods insert(CourseOfStudy), and loadRecordFromRs(ResultSet) in ManagerCourseOfStudy.

    • Modified method doGet(HttpServletRequest,HttpServletResponse) in class ServletInsertCourseOfStudy.

  • SMOS3 Add the field "Book" to the Teachings

    • New field book of type String in Teaching. New methods getBook(), and setBook(String) in the same class.

    • Modified methods insert(Teaching), update(Teaching), and loadRecordFromRs(ResultSet) in ManagerTeaching.

    • Modified method doGet(HttpServletRequest,HttpServletResponse) in class ServletInsertTeaching.

  • SMOS6 Allow students parents to visualize the school report, justification, absences, and disciplinary notes of their sons

    • New class ServletShowReportsParent extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing p request, and doing post.

    • New class ServletShowJustificationListToParents extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing p request, and doing post.

    • New class ServletShowNoteListToParents extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing p request, and doing post.

    • New class ServletShowRegisterToParents extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing p request, and doing post.

    • New class ServletLoadStudentForParent extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing post, and doing p request.

    • New class ServletAssignParentToStudent extending HttpServlet. This boundary class communicates mainly with HttpServletRequest, HttpServletResponse, and HttpSession. It allows doing p request, and doing post.

    • New field PARENT of type int in Role.

    • New methods getIdParent(), and setIdParent(int) in User.

    • New methods getParents(), isParent(User), assignParent(User,int), and removeParent(User) in ManagerUser.

    • Modified method doGet(HttpServletRequest,HttpServletResponse) in classes ServletAlterPersonalDate, ServletLogin, ServletShowUserRoleForm.

  • SMOS8 Allow the configuration of the SMOS parameters

    • New class ServletInitialize extending HttpServlet. This boundary class communicates mainly with ServletConfig, ServletException, and Utility, and consists mostly of mutator methods. It allows initing servlet config.

Bug Fixes

  • SMOS1 Assign Teaching to User does not show in alphabetic order

    • Modified method getTeachingsByClassroomId(int) in class ManagerTeaching.

  • SMOS5 DuplicateEntityException when inserting justification

    • Modified method exists(Justify) in class ManagerRegister.

  • SMOS7 User update does not work

    • Modified method doGet(HttpServletRequest,HttpServletResponse) in class ServletUpdateUser.

  • SMOS11 Votes should not be negative or higher than 10

    • Modified methods setWritten(int), setOral(int), and setLaboratory(int) in Votes.

Changes to Used Libraries

  • SMOS2.0 uses itext 2.0.8 instead of 1.4.8, and displaytag 1.3 instead of 1.1.
